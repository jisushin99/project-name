# 소일러(Soyler) 랜딩페이지 명세서

## 1. 프로젝트 개요
- **프로젝트명**: 소일러 랜딩페이지
- **유형**: 프리런칭 랜딩페이지 (이메일 수집)
- **핵심기능**: 위치 기반 마이크로잡 플랫폼 "소일러" 서비스 소개 및 이메일 수집
- **대상사용자**: 잠재적 서비스 이용자 (수행자/요청자 모두)

## 2. UI/UX 명세

### 레이아웃 구조
- **전체**: 단일 페이지 스크롤 방식
- **섹션**: 8개 섹션 (히어로, 문제정의, 솔루션, 작동방식, 마이크로잡예시, 비전, 이메일, 푸터)
- **반응형**: 모바일 (< 768px), 태블릿 (768px ~ 1024px), 데스크탑 (> 1024px)

### 시각 디자인

#### 색상 팔레트
- **Primary**: #4F46E5 (인디고)
- **Secondary**: #06B6D4 (시안)
- **Accent**: #F59E0B (앰버)
- **Background Light**: #F8FAFC
- **Background Gradient**: linear-gradient(135deg, #EFF6FF 0%, #E0F2FE 50%, #F0FDFA 100%)
- **Text Primary**: #1E293B
- **Text Secondary**: #64748B
- **Card Background**: #FFFFFF
- **Card Shadow**: 0 4px 20px rgba(79, 70, 229, 0.08)

#### 타이포그래피
- **Font Family**: 'Pretendard', 'Noto Sans KR', sans-serif
- **Hero Title**: 56px (데스크탑), 36px (모바일), font-weight: 800
- **Section Title**: 40px (데스크탑), 28px (모바일), font-weight: 700
- **Body**: 18px, font-weight: 400
- **Small**: 14px

####Spacing
- **Section Padding**: 100px 0 (데스크탑), 60px 0 (모바일)
- **Container Max Width**: 1200px
- **Card Padding**: 24px
- **Card Border Radius**: 20px

#### 시각 효과
- **Gradient Text**: Hero 타이틀에 그라디언트 텍스트 적용
- **Card Hover**: translateY(-8px), 그림자 증가
- **Floating Animation**:的手机Mockup에 floating 효과
- **Fade In Animation**: 스크롤 시 요소들 페이드인
- **Continuous Pulse**: CTA 버튼에 펄스 효과

### 컴포넌트

#### 1. 네비게이션
- 로고 (왼쪽): "소일러" 텍스트 로고
- CTA 버튼 (오른쪽): "출시 알림 받기"
- 스크롤 시 백그라운드 블러 효과 추가

#### 2. 히어로 섹션
- 배경: 부드러운 블루 그라디언트 + subtle 패턴
- 콘텐츠:
  -/badge: "📍 위치 기반 마이크로잡"
  - 타이틀: "당신 곁의 작은 일을\n새로운 기회로 연결하다"
  - 서브타이틀: "벌레 잡기, 줄 서주기, 택배 받기...\n언제 어디서나 필요한 순간, 가까운 이웃이 도와드려요"
  - 버튼 2개: "출시 알림 받기" (Primary), "서비스 알아보기" (Secondary)
- 오른쪽: 핸드폰 Mockup (애니메이션)

#### 3. 문제 정의 섹션
- 타이틀: "매일 조금씩 겪는\n불편함, 그대로放着-rest?"
- 카드 3개:
  1. "택배가 왔는데 집에 없어요"
  2. "긴 줄 대신 Somewhere Else에서...\n   줄 서주는 사람Needed"
  3. "무거운 짐 들어주세요~\n   도와줄 사람 없나요?"

#### 4. 솔루션 섹션
- 타이틀: "위치 기반,\n빠른 연결,\n신뢰가는 거래"
- 3개 피처:
  1. 🔵 Real-time 위치 추적
  2. ⚡ 5분 이내 매칭
  3. 💳 안전 결제 시스템

#### 5. 작동 방식 섹션
- 타이틀: "4STEP으로 시작하기"
- 스텝 카드 4개 (번호 표시):
  1. "프로필 등록" -自己的能力 소개
  2. "요청 발생" -필요한 순간 요청
  3. "근처 탐색" -가까운 수행자 찾기
  4. "빠른 수행" -작업 완료하고报酬

#### 6. 마이크로잡 예시 섹션
- 타이틀: "이런 일도 할 수 있어요"
- 카드 그리드 (2x3 or 3x2):
  1. 🐛 벌레 잡기
  2. 🚶 줄 서주기
  3. 📦 택배 수령
  4. 🏋️ 짐 옮기기
  5. 🛒 심부름
  6. 🌿garden Care

#### 7. 비전 섹션
- 긴 배경: 그라디언트
- 타이틀: "작은 일들이\n모이면 큰 시장이 됩니다"
- 서브: "소일러는 모든 '작은 도움'을\n새로운 일자리고 수입의 기회로 만들어요"

#### 8. 이메일 수집 섹션
- 타이틀: "출시 알림 받기"
- 서브: "가장 먼저 소일러를 경험해보세요"
- 이메일 입력창 + 버튼 (한 줄)
- 인라인 에러/성공 메시지

#### 9. 푸터
- 로고
- 소셜 링크 (선택적)
- copyright

### 핸드폰 Mockup 인터랙션
- 화면 내에서 프로필 카드, 위치 매칭, 요청 화면이 순환하는 애니메이션
- CSS animation으로 구현 (페이드인/아웃 + 슬라이드)
- 각 화면 3초씩 표시

## 3. 기능 명세

### 핵심 기능
1. 스크롤 시 섹션별 페이드인 애니메이션
2. 이메일 입력 및 유효성 검증
3. 이메일 제출 (로컬 스토리지 또는 콘솔 출력)
4. 네비게이션 스크롤 시 배경变化
5. 카드 호버 효과
6. 핸드폰 화면 슬라이드쇼

### 사용자 상호작용
- CTA 버튼 클릭 → 이메일 섹션으로 스크롤
- 이메일 입력 후 제출 → 성공 메시지 표시
- 카드 호버 → 시각적 피드백

### 에지 케이스
- 이메일이 유효하지 않을 때: 에러 메시지 표시
- 이메일 중복 제출: 중복 안내 메시지
